# Claude Code Demo

Interactive demo showcasing Claude Code's capabilities.

## Features

- File operations and code editing
- Command execution and project management  
- Code search and analysis
- Git integration and more

## Usage

```bash
python3 demo_app.py
```

## Claude Code Integration

This project demonstrates how Claude Code can help with:
- Creating and managing code projects
- Running tests and builds
- Git operations and GitHub integration
- VS Code development workflow
